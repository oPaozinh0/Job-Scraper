# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in Job Scraper, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### How to Report

Please send an email to **davioliveira353.do@gmail.com** with the subject "Security Vulnerability - Job Scraper".

In your email, please include:

* A description of the vulnerability.
* Steps to reproduce the issue.
* The potential impact of the vulnerability.

### Our Response

We are committed to addressing security vulnerabilities promptly.
* We will acknowledge receipt of your report within 48 hours.
* We will investigate the issue and determine its impact.
* We will work on a fix and release a security update as soon as possible.
* We will keep you informed of our progress.

Thank you for helping us keep Job Scraper secure!
